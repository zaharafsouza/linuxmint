#06_ Teclas de Atalho do Terminal
a) Ctrl + Alt + T: Terminal
b) Ctrl + D: Comando exit
c) Ctrl + Alt + Backspace: Finalizar ambiente gráfico
d) Ctrl + Alt + F? (F1 até F7): Bash/Shell (puro)
F1 até F6: Terminal
F7: Ambiente Gráfico 
e) Ctrl + L: clear
f) TAB (tabulador): Auto-Complemento
g) Ctrl + Alt + L: Bloquear a Tela

#07_ Atualizando o Sistema via Terminal

a) Linux Case Sensitive
b) O que é opções e parâmetros dos comandos Linux

sudo -i

O que é sources.list

https://guiafoca.org/guiaonline/inicianteintermediario/ch20s02.html

#Atualizar as listas do Sources.list
#01) sudo apt update

#Atualizar todos os softwares
#02) sudo apt upgrade

#Atualizar todos os softwares Forçado
#03) sudo apt full-upgrade

#Atualizar todo o Sistema Operacional (Kernel - Núcleo)
#04) sudo apt dist-upgrade

#Limpar aplicativos desnecessários
#05) sudo apt autoremove

#Limpar o sistema operacional
#06) sudo apt autoclean

#Limpar as Lista do Sources.List
#07) sudo apt clean
#08) sudo apt update
#09) sudo reboot
